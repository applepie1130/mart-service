# 마트 서비스 (v0.0.1 개발진행중)
## 1. 서비스 소개
- 매번 기억하기 어려운 대형마트 휴일정보. 이제는 간단히 통합된 마트 정보를 확인 할 수 있습니다.
- 근처에 있는 모든 마트나 특정 대형마트(이마트, 롯데마트, 홈플러스, 코스트코)의 휴일정보, 기본 마트정보를 확인 할 수 있습니다.

## 2. 주요기능
- 근처 대형마트의 휴일정보 제공
- 특정 대형마트의 휴일정보 제공
- 마트정보
  - 대형마트의 휴일정보 제공
  - 사용자의 현재위치와 해당마트와의 거리정보 지도제공
  - 로케이션 정보 제공
